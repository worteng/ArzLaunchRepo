# ArzLaunchRepo

Публичный склад скриптов, тем и модов для **Arizona RP Launcher**.
Всё содержимое этого репозитория интегрировано прямо в лаунчер — пользователи могут устанавливать файлы одним кликом без ручного скачивания.

---

## 📁 Структура репозитория

```
ArzLaunchRepo/
├── Lua/          # MoonLoader Lua-скрипты
├── Themes/       # Темы интерфейса лаунчера
└── ...           # Другие категории в будущем
```

---

## 🌙 Lua — MoonLoader скрипты

Папка `Lua/` содержит скрипты для [MoonLoader].

### Формат записи в `lua.txt`
Каждый скрипт описывается блоком:

```
[script]
name        = Название скрипта
author      = Автор
description = Краткое описание что делает скрипт
usage       = Как пользоваться (клавиши, команды и т.п.)
file        = ИмяФайла.lua
url         = https://raw.githubusercontent.com/.../Lua/ИмяФайла.lua
icon        = 🌙
tag         = HUD
```

**Доступные теги:** `HUD`, `Camera`, `QoL`, `Visual`, `Utils`, `RP`, `Other` и т.п.

## 🎨 Themes — темы интерфейса

Папка `Themes/` содержит JSON-файлы с цветовыми схемами лаунчера.

### Формат файла темы (`НазваниеТемы.json`)

```json
{
  "name": "Название темы",
  "author": "Автор",
  "description": "Краткое описание",
  "accent":   [0, 120, 212],
  "accent2":  [0, 180, 255],
  "surface":  [11, 13, 20]
}
```

Значения — RGB от 0 до 255. `accent` используется для кнопок и рамок, `accent2` — для переключателей и подсветки, `surface` — основной фон панелей.

### Формат записи в `themes.txt`

```
[theme]
name        = Название темы
author      = Автор
description = Краткое описание
file        = НазваниеТемы.json
url         = https://raw.githubusercontent.com/.../Themes/НазваниеТемы.json
icon        = 🎨
preview     = https://raw.githubusercontent.com/.../Themes/previews/НазваниеТемы.png
```

## 📬 Как отправить свой скрипт или тему

1. Открой **Arizona RP Launcher**
2. Найди кнопку **«Поделиться с комьюнити»**
3. Откроется Telegram-бот — отправь туда свой `.lua` файл или `.json` тему
4. После проверки автором репозитория файл будет опубликован здесь

> Бот: **@ARZLaunchBot**

---

## ✅ Требования к публикации

**Для Lua-скриптов:**
- Файл должен запускаться без ошибок в актуальной версии MoonLoader
- Не должен содержать вредоносного кода, читов и другой хрени
- Должно быть описание и хотя бы базовая инструкция по использованию

**Для тем:**
- JSON должен содержать все три обязательных поля: `accent`, `accent2`, `surface`
- Рекомендуется приложить скриншот-превью (PNG, ~400×240px)
